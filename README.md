# CTSLibraryJ

## Description

The CTSLibraryJ is a resources library to conversion, validation and formatting.

## Features of Library

* Use Apache Maven as manager build
* Javadoc tool to generate library documentation
* Unit tests with JUnit framework

## Resources of Library

+ Conversion of values
+ Data manipulation
+ Extenso
+ Formatting of values
+ Validation of values
