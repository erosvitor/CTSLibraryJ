# CTSLibraryJ

The **CTSLibraryJ** is a project that contens several routines to manipulation of strings and numbers.

* Apache Maven 3.6.3
* Javadoc
* JUnit 4.12

+ Data manipulation
+ Extenso
+ Formatting of values
+ Validation of values
